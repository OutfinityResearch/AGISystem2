Ontology

Back to index | Back to concepts

Ontology in philosophy is the study of what exists; here it designates the part of the vector space that encodes factual structure about entities, properties and relations, kept distinct from evaluative or normative information.

In AGISystem2

The system maintains a clear separation between ontological (factual) and axiological (value-based) dimensions in the vector space. This separation ensures that factual representations remain pure and uncontaminated by value judgements.

Key ontological concepts include:

  • Ontology dimensions - Axes that encode factual information about the world
  • Entity representation - How entities are encoded as points in the space
  • Property encoding - How properties and attributes are represented
  • Relational structure - How relationships between entities are encoded

Technical Implementation

Ontological reasoning is managed through the VectorSpace module, which provides operations for creating and manipulating factual representations. The system uses geometric operations to reason about entities, properties, and their relationships while maintaining the separation from value-based dimensions.
