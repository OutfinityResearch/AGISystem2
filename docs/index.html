What is AGISystem2?

AGISystem2 is an AI research platform inspired by the System 2 concept from cognitive psychology—the slow, deliberate, logical mode of human reasoning (as opposed to System 1's fast, intuitive responses). Born from European research initiatives, the platform brings together formal verification, neural-symbolic hybrid reasoning, and hyperdimensional computing.

A key innovation is the introduction of a semantic programming language (DSL) that gives formal, machine-verifiable meaning to natural language. Instead of teaching humans to write in formal logic, we created a language that captures the semantic structure implicit in everyday language. The result is natural language with mathematical guarantees.

The Core Innovation: Semantic Libraries

Just as programming languages have libraries for math, networking, or graphics, AGISystem2 enables semantic libraries for any domain—medicine, law, physics, biology. Each library defines:

  • Vocabulary: Domain-specific terms and their formal meaning
  • Rules: How concepts relate and what follows from what
  • Inference: Automatic reasoning over the domain's knowledge

"Instead of teaching humans to speak to machines in formal languages, we teach machines to understand the formal structure implicit in human language."

Why This Matters

For applications where verification is critical—compliance, medical decisions, legal reasoning, safety-critical systems—probabilistic pattern matching alone isn't enough. AGISystem2 provides:

  • Complete audit trails: Every conclusion traces back to facts and rules
  • Contradiction detection: The system flags inconsistencies in knowledge
  • Reproducible results: Same input always produces same output
  • Domain modularity: Semantic libraries can be composed and reused

Scope & Limitations

What AGISystem2 does well:

  • Taxonomic reasoning (IS_A hierarchies, transitive inference)
  • Rule-based inference with backward chaining
  • Explicit contradiction detection within formalized domains
  • Complete audit trails for every conclusion

What AGISystem2 does NOT do:

  • Full natural language understanding—includes a hardcoded English NL→DSL translator for common patterns; LLMs can translate other languages into our supported English format (less flexible, more formalizable)
  • Creative generation or open-ended conversation
  • Learning from unstructured data
  • Handling domains that haven't been formally modeled

Fundamental limitation: The system reasons correctly within a theory, but cannot guarantee the theory itself is correct. See The Symbol Grounding Problem for an honest discussion of this epistemological limitation.

Philosophy: Theories as Tools

AGISystem2 adopts a pragmatic stance: theories are tools, not truths. The system guarantees correct reasoning within a theory, not the correctness of the theory itself. This isn't a limitation—it's how all formal systems work, from mathematics to legal codes.

The value is practical: transform natural language specifications into formal theories that can be analyzed, debugged, and revised. The consequences of your knowledge become computable, verifiable, and auditable.
