Dense Binary Strategy

Home · Architecture · Theory · Syntax · APIs · Wiki · Specs Default HDC implementation for AGISystem2

The Dense Binary Strategy is the default HDC implementation in AGISystem2, using compact binary vectors with XOR and majority operations for maximum efficiency and determinism.

Overview

✓ Strengths

  • Fastest operations (O(geometry/64))
  • Minimal memory usage (4KB per vector)
  • Fully deterministic results
  • Simple implementation
  • Optimal for real-time applications

✗ Limitations

  • Limited capacity (~200 facts)
  • No semantic clustering
  • Basic noise tolerance
  • Fixed geometry (32,768 bits)

Core Operations

1. BIND (XOR) - Association

Bind Operation: C = A ⊕ B
Bind Operation Properties: Commutative: A ⊕ B = B ⊕ A Associative: (A ⊕ B) ⊕ C = A ⊕ (B ⊕ C) Self-inverse: A ⊕ A = 0 (zero vector) Reversible: (A ⊕ B) ⊕ B = A Identity: A ⊕ 0 = A Implementation: Bitwise XOR on 512 × 64-bit words Complexity: O(geometry/64) = O(512) operations Deterministic: Yes (bitwise operation)

2. BUNDLE (Majority Vote) - Superposition

Bundle Operation: Majority([A, B, C, ...])
Bundle Operation Properties: Input: [A, B, C, D, ...] (n vectors) Output: C where C[i] = majority(A[i], B[i], C[i], ...) Properties: • Preserves similarity to all inputs • Non-reversible (cannot extract individual items) • Capacity limited (~200 items before saturation) • Preserves vector extension compatibility Implementation: Bitwise majority vote on 512 × 64-bit words Complexity: O(n × geometry/64) = O(n × 512) Deterministic: Yes (majority function)

3. SIMILARITY - Comparison

Similarity Operation: Hamming Distance
Similarity Operation Properties: Formula: similarity(A, B) = 1 - (hamming_distance(A, B) / dimension) Where: hamming_distance(A, B) = popcount(A ⊕ B) popcount = number of 1 bits Interpretation: • 1.0: Identical vectors • 0.5: Unrelated (random) • 0.0: Inverse (bitwise NOT) Implementation: XOR + popcount on 512 × 64-bit words Complexity: O(geometry/64 + popcount) = O(512 + sum) Deterministic: Yes (XOR + count)

Implementation Details

// Vector representation type Vector = Uint32Array[1024]; // 1024 × 32 bits = 32,768 bits // Bind operation (XOR) function bind(a: Vector, b: Vector): Vector { const result = new Uint32Array(1024); for (let i = 0; i < 1024; i++) { result[i] = a[i] ^ b[i]; } return result; } // Bundle operation (Majority Vote) function bundle(vectors: Vector[]): Vector { const result = new Uint32Array(1024); const count = new Uint32Array(1024); for (const vec of vectors) { for (let i = 0; i < 1024; i++) { count[i] += popcount(vec[i]); } } for (let i = 0; i < 1024; i++) { result[i] = count[i] > (vectors.length / 2) ? 0xFFFFFFFF : 0; } return result; } // Similarity operation function similarity(a: Vector, b: Vector): number { let distance = 0; for (let i = 0; i < 1024; i++) { distance += popcount(a[i] ^ b[i]); } return 1 - (distance / 32768); }
