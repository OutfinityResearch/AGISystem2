Verifiable Computing & ZKPs

Proving that AI computation is correct without compromising privacy.

What is Verifiable Computing?

Verifiable computing allows a client to outsource the computation of a function to an untrusted server, while receiving a proof that the computation was performed correctly. In the AI era, this is known as zkML (Zero-Knowledge Machine Learning).

Core Technologies

  • ZK-SNARKs: Succinct Non-interactive Arguments of Knowledge. These are tiny proofs that can verify massive AI computations in milliseconds.
  • Fully Homomorphic Encryption (FHE): Allowing AI to run directly on encrypted data.

Strategic Value

Auditability is a pillar of AGISystem2. By using zkML, we can provide a cryptographic guarantee that an agent followed its formal specifications and utilized only approved evidence, without the auditor needing access to the proprietary data or models.
