Automated Theorem Proving (ATP)

Turning AI from a "guesser" into a "verifier" of mathematical truth.

What is ATP?

Automated Theorem Proving is the endeavor to prove mathematical theorems by computer programs. In the AI era, this has evolved from pure brute-force search to a synergistic partnership between LLMs and formal proof assistants.

Core Proof Assistants

  • Lean: A modern theorem prover and programming language developed at Microsoft Research. It is currently the primary target for AI-for-math initiatives (e.g., Google DeepMind's AlphaGeometry).
  • Coq: A formal proof management system that allows the expression of mathematical assertions and mechanically checks proofs of these assertions.
  • Isabelle/HOL: A generic proof assistant used for formalizing mathematics and verifying software.

AI + Formal Proofs

We are tracking the trend of using LLMs to generate Formal Proof Scripts. Because proof assistants like Lean are "merciless" verifiers, the LLM receives immediate feedback: if the script compiles, the reasoning is guaranteed to be correct.

Impact on AGISystem2

ATP is the "hard gate" of our System 2 vision. For high-stakes enterprise decisions, we aim to have the agent output not just a recommendation, but a Machine-Verified Proof that the recommendation follows from the provided evidence and constraints.
