Geometric Deep Learning

Unifying the world of AI through symmetry and invariance.

What is Geometric Deep Learning?

Geometric Deep Learning (GDL), spearheaded by Michael Bronstein, is a framework that unifies diverse neural network architectures (like CNNs, GNNs, and Transformers) through the lens of symmetry groups and invariance. It posits that the structure of the data domain should dictate the architecture of the network.

Core Principles

  • Invariance: A property where the output doesn't change even if the input is transformed (e.g., rotating an image).
  • Equivariance: A property where the transformation of the input results in a predictable transformation of the output.
  • Graph Neural Networks (GNNs): Operating on non-Euclidean data like social networks or chemical compounds.

Relevance to AGISystem2

We use GDL to build Knowledge Graph Embeddings that respect the topological structure of facts. This allows our agents to perform sophisticated relational reasoning (System 2) that is robust to structural permutations.
