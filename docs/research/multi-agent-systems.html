Multi-Agent Systems (MAS)

Orchestrating intelligence through formal coordination and consensus.

What is a Multi-Agent System?

A multi-agent system is a computerized system composed of multiple interacting intelligent agents. In our research, we move from "one large model" to "many specialized agents" that must coordinate to solve complex problems.

Key Coordination Mechanisms

  • Consensus Protocols: Algorithms that allow agents to agree on a single state of the world (e.g., Paxos, Raft).
  • Agent Communication Languages (ACL): Using structured protocols (like FIPA-ACL or our own CNL-based languages) to exchange intentions.
  • Market-Based Coordination: Using economic principles (bidding, auctions) to allocate resources and tasks between agents.

Epistemic Redundancy

MAS allows for "System 2" robustness at the network level. By having different agents assume different roles (e.g., a "Generator" and an "Adversarial Verifier"), we can surface disagreements and errors that a single model would miss.
