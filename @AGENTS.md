# Agent Instructions for AGISystem2

## Article Writing Style (MRP Section)

When writing or updating articles for the Meta Rational Pragmatics (MRP) section, adhere to the following standards:

1.  **Language:** English.
2.  **Structure:** Use the provided `article-template.html` (or follow existing articles like `meta-rationality-ai-scientific-inquiry.html`).
3.  **Sections:** Use short, descriptive section titles. Each section should have a clear ID for the TOC.
4.  **Citations:** Use standard bracketed citations (e.g., `[FRUGALGPT-2023]`). Include a `References` section at the end of the article with IDs matching the in-text citations.
5.  **Metadata:** Update the `<title>`, `<meta name="description">`, Open Graph tags, kicker (e.g., `MRP Article XX`), and subtitle.
6.  **TOC:** Ensure the Table of Contents (aside element) matches the sections in the main article.
7.  **Index Update:** When adding a new article, update `docs/MRP/index.html` by incrementing the article count and adding the article card to the list.
8.  **Tone:** Professional, restrained, and analytical. Avoid rhetorical finality.
